🚀 DNS-over-HTTPS 代理服务

这是一个基于 Cloudflare Pages Functions 的 DNS-over-HTTPS (DoH) 代理服务。

📡 API 端点

URL: /dns-query

方法: GET 或 POST

内容类型: application/dns-message

💡 使用示例

GET 请求:

curl "https://your-domain.pages.dev/dns-query?dns=AAABAAABAAAAAAAAA3d3dwdleGFtcGxlA2NvbQAAAQAB"

POST 请求:

curl -X POST "https://your-domain.pages.dev/dns-query" \
  -H "Content-Type: application/dns-message" \
  --data-binary @dns-query.bin

🛠️ 支持的上游服务器

  • Google DNS (dns.google)
  • Cloudflare DNS (cloudflare-dns.com)

✨ 特性

  • 🔄 自动故障转移
  • 🚀 全球 CDN 加速
  • 🔒 HTTPS 加密
  • 🌍 CORS 支持
  • ⚡ 智能缓存
